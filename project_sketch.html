Project Sketch

Planned Tools

The data is relatively clean and has already been scaled to percentages. The only preprocessing that may be necessary will be renaming the columns for clarity (I can do this from the command line). However, I may elect to use annual GDP data by country from an external source. In that case, I will use Python (specifically pandas) to facilitate any necessary merging. In terms of visualization, I will be using D3 exclusively.

Planned Techniques

  • My first visualization is inspired by Hans Rosling and will be my most ambitious. In particular, I will display the data as a bubble chart, with each bubble representing an individual country. Because the data also represents a time series within each country, I will have the points move along the x-y coordinate plane over time. Instead of choosing which two features to plot against on the x- and y-axes, I will allow the user to interactively choose which features to display. Not only will this give me the opportunity to learn how to deal with a time component, it will also force me to learn how to dynamically adjust the x- and y-axes of a plot based on user input. If I have the opportunity to merge the appropriate data, I will also scale each bubble by the country's GDP.
  • My second visualization will be a parallel coordinate plot that will change over time to reflect the changing years. Although this does not provide as granular of a view per country, it does make it easier to spot any sorts of trends among the variables in the data. Similar to the previous technique, I hope to learn how to implement smooth transitions over time, this time in a parallel coordinate setting.
  • My third visualization will be a multi-line plot to visualize the time series, so the x-axis will be the date. The user will have the option to dynamically choose what feature the y-axis should be. Each line in the multi-line chart will represent an individual country, so I will also implement a tooltip that will display the name of the corresponding country on hover. From this visualization, I hope to learn how to work with tooltips in line charts (specifically, how to represent the entire line as a single entity on hover).

Planned Interaction

The most important feature in my bubble chart and multi-line plot will be the user's ability to select what to display on the x- and y-axes, perhaps via a drop-down menu. This implies that I will also need to implement smooth transitions when the user changes axis selections.

Another important feature will be the use of time in the bubble chart and parallel coordinate plot. Not only will I need to implement smooth transitions between time periods, I would also like to allow the user to specify a particular point in time at which he/she wants to observe the data (in effect stopping the transitions over time). This might be in the form of a time scale with a pointer that moves across the time scale over time. When selected, the user can specify the point he/she wants to see a "snapshot" of the data.

Other features will include brushing for the parallel coordinate plot and mouseover tooltips for the bubble chart and multi-line plot.

Planned Interface

Bubble Chart

  • The user can select the x-axis, y-axis, and specific point in time at which the points should be displayed
  • This will be the only plot in which the different observations have different colors (similar to Hans Rosling's demonstration)
  • Note that, by default, the points will move automatically through the years
  • Although it isn't shown in the sketch, there will also be a tooltip feature to display the country name for each point

Parallel Coordinate Plot

  • All 9 features will be displayed in the parallel plot
  • The user can select which point in time he/she wants to see the data. Otherwise, the data will transition from year to year

Multi-line Plot

  • Here, the user will be able to select what to display on the y-axis
  • There will also be a tooltip on mouseover showing the country name for each observation
